# ai-service-desk-mvp
agentkit-internal-support 是一个基于 OpenAI AgentKit 构建的企业内部智能客服（Service Desk）MVP 项目。  本项目展示了如何使用 Agentic Workflow 将大模型与企业内部知识库和业务系统结合，实现从「问题理解 → 知识检索 → 工单创建 → 状态反馈」的完整闭环。  项目目标不是做一个“聊天机器人”，而是提供一个可学习、可扩展、可落地的企业级 Agent 架构示例，适用于：  企业内部 IT / HR / 行政 / 财务支持  智能工单与服务台系统  AgentKit / Agentic AI 实践学习
